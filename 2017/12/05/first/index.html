使用Hexo快速搭建个人博客

这个博客也是我第一次搭建，使用Hexo加GitHub搭建。比较简单记录一下过程吧。

1.创建Github 域名和空间

1.1注册

首先注册一个GitHub账号，你注册时填写的UserName将是你的搭建后的域名空间。如:https://hankmax.github.io/

记住这个username，以下所有username全替换成自己创建的username

1.2创建博客托管仓库

登陆你的github进入首页，点击New repository创建新仓库

注意username要和你注册的username填写一致

2.安装Hexo

前置条件

  • 已安装git
  • 已安装node

打开命令行，全局安装Hexo

3.创建配置Hexo项目

3.1初始化Hexo项目

成功过后就有一个当前目录就会有个名为username.github.io的文件夹

3.2 基础配置

主题安装

Hexo提供很多免费主题供你使用，在主题选择一个。克隆到你的项目目录中

项目配置

打开username.github.io/_config.yml文件修改基础配置

注意repo填写你在github创建的仓库的地址

修改配置冒号后跟一个空格不然无法配置成功

主题配置

主题配置文件在username.github.io/themes/next/_config.yml
具体配置可看各主题的文档进行配置，本主题配置文档

写文章

新建markdown文件，放到username.github.io/source/_posts目录下

First.md

测试

$hexo s
在项目目录下启动测试服务，打开https://localhost:4000 访问

4.发布到github

1.安装hexo-deployer-git自动部署发布工具

$npm install hexo-deployer-git --save

2.进行部署

$hexo clean && hexo g && hexo d

3.稍等片刻，打开https://username.github.io访问

Runtime总结

  • 站点概览
